## 🚀 How to Run

Clone the repository:

```bash
git clone <your-repository-link>
```

Navigate to the project folder:

```bash
cd Python-Beginner-Projects
```

Run any project:

```bash
python filename.py
```

Example:

```bash
python Rockpaperscissor.py
```
